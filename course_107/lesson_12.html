Select (ﺍﻧﺘﺨﺎﺏ ﮐﺮﺩﻥ): ﺍﮔﺮ ﻣﯽ‌ﺧﻮﺍﻫﻴﺪ ﺁﻳﺘﻤﯽ ﺭﺍ ﺑﺮ ﺭﻭﯼ ﺩﺳﮑﺘﺎﭖ ﺍﻧﺘﺨﺎﺏ ﮐﻨﻴﺪ، ﺳﻌﯽ ﮐﻨﻴﺪ ﺍﻳﻦ ﮐﺎﺭ ﺭﺍ ﺑﺎ ﻧﻮﮎ ﺍﻧﮕﺸﺖ ﺧﻮﺩ ﺍﻧﺠﺎﻡ ﺩﻫﻴﺪ ﺯﻳﺮﺍ ﻣﻤﮑﻦ ﺍﺳﺖ ﺳﻄﺢ ﺍﻧﮕﺸﺖ‌ﺗﺎﻥ ﺑﺰﺭﮒ ﺑﺎﺷﺪ (ﻣﮕﺮ ﺍﻳﻨﮑﻪ ﺩﺳﺘﺎﻥ ﮐﻮﭼﮑﯽ ﺩﺍﺷﺘﻪ ﺑﺎﺷﻴﺪ). 
 
ﺩﺑﻞ ﮐﻠﻴﮏ: ﺩﺭ ﺍﻳﻨﺠﺎ ﻧﻴﺰ ﺍﺳﺘﻔﺎﺩﻩ ﺍﺯ ﻧﻮﮎ ﺍﻧﮕﺸﺖ‌ﺗﺎﻥ ﻣﻨﺎﺳﺐ ﺍﺳﺖ. ﺑﺮﺍﯼ ﺩﺑﻞ ﮐﻠﻴﮏ ﮐﺮﺩﻥ ﻳﮏ ﺁﻳﺘﻢ ﺑﺮ ﺭﻭﯼ ﺩﺳﮑﺘﺎﭖ، ﺩﻭ ﺑﺎﺭ ﺑﺮ ﺭﻭﯼ ﺁﻥ ﺿﺮﺑﻪ ﺑﺰﻧﻴﺪ. 
 
ﺭﺍﺳﺖ ﮐﻠﻴﮏ: ﺍﻳﻦ ﮐﺎﺭ ﮐﻤﯽ ﭘﻴﭽﻴﺪﻩ‌ﺗﺮ ﺍﺳﺖ. ﺑﺮﺍﯼ ﺭﺍﺳﺖ ﮐﻠﻴﮏ ﮐﺮﺩﻥ ﺑﺮ ﺭﻭﯼ ﻳﮏ ﺁﻳﺘﻢ، ﺍﺑﺘﺪﺍ ﺑﻪ ﺁﺭﺍﻣﯽ ﻧﻮﮎ ﺍﻧﮕﺸﺖ ﺧﻮﺩ ﺭﺍ ﺑﺮ ﺭﻭﯼ ﺻﻔﺤﻪ ﻗﺮﺍﺭ ﺩﻫﻴﺪ ﻭ ﺻﺒﺮ ﮐﻨﻴﺪ ﺗﺎ ﻣﺮﺑﻊ ﮐﻮﭼﮑﯽ ﺑﺮ ﺭﻭﯼ ﺻﻔﺤﻪ ﻧﻤﺎﻳﺶ ﻇﺎﻫﺮ ﺷﻮﺩ. ﺍﻳﻦ ﻣﻨﻮ، ﺣﺘﯽ ﺍﮔﺮ ﺍﻧﮕﺸﺖ ﺧﻮﺩ ﺭﺍ ﺍﺯ ﺭﻭﯼ ﺻﻔﺤﻪ ﺑﺮﺩﺍﺭﻳﺪ، ﺑﺮ ﺭﻭﯼ ﺻﻔﺤﻪ ﺑﺎﻗﯽ ﻣﯽ‌ﻣﺎﻧﺪ. ﺳﭙﺲ ﻣﯽ‌ﺗﻮﺍﻧﻴﺪ ﮔﺰﻳﻨﻪ ﻣﻮﺭﺩ ﻧﻈﺮ ﺧﻮﺩ ﺭﺍ ﺍﺯ ﺑﻴﻦ ﮔﺰﻳﻨﻪ‌ﻫﺎﯼ ﻣﻮﺟﻮﺩ ﺩﺭ ﺍﻳﻦ ﻣﻨﻮ ﺍﻧﺘﺨﺎﺏ ﮐﻨﻴﺪ.
 
ﺩﻭﺑﺎﺭﻩ ﺗﺎﮐﻴﺪ ﻣﯽ‌ﮐﻨﻴﻢ ﮐﻪ ﺑﻬﺘﺮﻳﻦ ﺭﻭﺵ ﺑﺮﺍﯼ ﮐﺎﺭ ﺑﺎ ﺩﺳﺘﮕﺎﻩ‌ﻫﺎﯼ ﻟﻤﺴﯽ ﺗﺤﺖ ﻭﻳﻨﺪﻭﺯ ۱۰ ﺍﻳﻦ ﺍﺳﺖ ﮐﻪ ﻣﻮﺱ ﻭ ﺻﻔﺤﻪ ﮐﻠﻴﺪﯼ ﺧﺮﻳﺪﺍﺭﯼ ﮐﻨﻴﺪ ﮐﻪ ﺍﺯ ﻃﺮﻳﻖ ﺑﻠﻮﺗﻮﺙ ﺑﻪ ﺳﻴﺴﺘﻢ ﻣﺘﺼﻞ ﻣﯽ‌ﺷﻮﻧﺪ. ﺩﺭ ﺗﺒﻠﺖ‌ﻫﺎﯼ ﮐﻮﭼﮏ‌ﺗﺮ ﺍﮔﺮ ﺑﺨﻮﺍﻫﻴﺪ ﺍﺯ ﺩﺳﺖ ﺧﻮﺩ ﺑﺮﺍﯼ ﮐﻨﺘﺮﻝ ﺩﺳﮑﺘﺎﭖ ﺍﺳﺘﻔﺎﺩﻩ ﮐﻨﻴﺪ ﮐﺎﺭﺗﺎﻥ ﺳﺨﺖ‌ﺗﺮ ﻣﯽ‌ﺷﻮﺩ. ﺍﻟﺒﺘﻪ ﮐﺎﺭﮐﺮﺩﻥ ﺗﺤﺖ ﭼﻨﻴﻦ ﺷﺮﺍﻳﻄﯽ ﺍﻣﮑﺎﻥ‌ﭘﺬﻳﺮ ﺍﺳﺖ ﻭﻟﯽ ﺍﮔﺮ ﺍﺯ ﻣﺤﻴﻂ ﺩﺳﮑﺘﺎﭖ ﺯﻳﺎﺩ ﺍﺳﺘﻔﺎﺩﻩ ﻣﯽ‌ﮐﻨﻴﺪ، ﺑﺮﺍﯼ ﻣﺎﻧﻮﺭ ﻭ ﮐﻨﺘﺮﻝ ﺑﻬﺘﺮ ﺑﺮ ﺭﻭﯼ ﺍﻳﻦ ﺭﺍﺑﻂ ﮐﺎﺭﺑﺮﯼ ﻗﺪﻳﻤﯽ، ﺑﻬﺘﺮ ﺍﺳﺖ ﻣﻮﺱ ﻭ ﺻﻔﺤﻪ ﮐﻠﻴﺪ ﺗﻬﻴﻪ ﮐﻨﻴﺪ.
 
ﺗﻐﻴﻴﺮ ﻭﺿﻮﺡ (ﺭﺯﻭﻟﻮﺷﻦ) ﺻﻔﺤﻪ ﻧﻤﺎﻳﺶ

ﺍﺛﺮ ﺗﻐﻴﻴﺮ ﻭﺿﻮﺡ ﺑﺮ ﺭﻭﯼ ﺍﻧﺪﺍﺯﻩ ﺁﻳﮑﻮﻥ‌ﻫﺎ، ﻏﻴﺮﻣﺴﺘﻘﻴﻢ ﺍﺳﺖ ﺯﻳﺮﺍ ﺩﺭ ﻭﻳﻨﺪﻭﺯ ۱۰ ﻫﺮ ﺁﻳﮑﻮﻥ ﻭ ﻫﺮ ﮐﺎﺭﺍﮐﺘﺮِ ﻣﺘﻨﯽ ﺑﻪ ﺻﻮﺭﺕ ﺗﻌﺪﺍﺩِ ﺍﺯ ﭘﻴﺶ ﺗﻌﺮﻳﻒ ﺷﺪﻩ‌ﺍﯼ ﭘﻴﮑﺴﻞ ﺑﻪ ﺍﺯﺍﯼ ﻭﺿﻮﺡ ﺧﺎﺹ ﺗﻌﺮﻳﻒ ﺷﺪﻩ ﺍﺳﺖ. ﻣﺜﻼ ﻣﻤﮑﻦ ﺍﺳﺖ ﻳﮏ ﺁﻳﮑﻮﻥ ﺧﺎﺹ ﺍﺑﻌﺎﺩﯼ ﻣﻌﺎﺩﻝ ۲۳ ﭘﻴﮑﺴﻞ ﺩﺍﺷﺘﻪ ﺑﺎﺷﺪ، ﻭﻟﯽ ﺑﺮ ﺭﻭﯼ ﻣﺎﻧﻴﺘﻮﺭﻫﺎﯼ ﻣﺨﺘﻠﻒ ﺑﻪ ﺷﮑﻞ‌ﻫﺎﯼ ﻣﺨﺘﻠﻒ ﻧﺸﺎﻥ ﺩﺍﺩﻩ ﺷﻮﺩ. ﺍﻳﻦ ﻧﺴﺒﺖ ﻣﻤﮑﻦ ﺍﺳﺖ ﺑﻪ ﺻﻮﺭﺕ ﻳﮏ ﺑﻴﺴﺘﻢ ﺻﻔﺤﻪ ﻧﻤﺎﻳﺶ ﺗﺒﻠﺖ ﻳﺎ ﻳﮏ ﭼﻬﻠﻢ ﺻﻔﺤﻪ ﻧﻤﺎﻳﺶ ﻳﮏ ﻣﺎﻧﻴﺘﻮﺭ ﺑﺰﺭﮒ ﺑﺎﺷﺪ. ﺑﻪ ﻧﻮﻋﯽ، ﻭﺿﻮﺡ ﻋﻮﺍﻗﺒﯽ ﻫﻢ ﺩﺭ ﭘﯽ ﺩﺍﺭﺩ ﺯﻳﺮﺍ ﻣﻤﮑﻦ ﺍﺳﺖ ﺑﺮ ﺭﻭﯼ ﺻﻔﺤﺎﺕ ﻧﻤﺎﻳﺶ ﺑﺎ ﻭﺿﻮﺡ ﺑﺎﻻ ﺩﺭ ﺩﻳﺪﻥ ﺁﻳﮑﻮﻥ‌ﻫﺎ ﻭ ﻣﺘﻮﻥ ﺑﺎ ﺍﻧﺪﺍﺯﻩ ﮐﻮﭼﮏ ﺩﭼﺎﺭ ﻣﺸﮑﻞ ﺷﻮﻳﺪ. ﺗﻐﻴﻴﺮ ﻭﺿﻮﺡ ﺗﺼﻮﻳﺮ ﮐﺎﺭ ﺧﻴﻠﯽ ﭘﻴﭽﻴﺪﻩ‌ﺍﯼ ﻧﻴﺴﺖ.

ﺩﺭ ﻣﺮﻭﺭﮔﺮ ﻓﺎﻳﺮﻓﺎﮐﺲ، ﻫﻤﻴﻦ ﻣﺮﺍﺣﻞ ﻗﺒﻞ ﺭﺍ ﺩﻧﺒﺎﻝ ﮐﺮﺩﻩ ﻭ ﺳﭙﺲ ﮔﺰﻳﻨﻪ Set Desktop Background (ﺗﻨﻈﻴﻢ ﭘﺲ ﺯﻣﻴﻨﻪ ﺩﺳﮑﺘﺎﭖ) ﺭﺍ ﺍﻧﺘﺨﺎﺏ ﮐﻨﻴﺪ. ﺑﺎ ﺍﻳﻦ ﮐﺎﺭ ﻳﮏ ﺩﻳﺎﻟﻮﮒ ﺑﺎﮐﺲ ﺑﺎﺯ ﺧﻮﺍﻫﺪ ﺷﺪ ﮐﻪ ﺍﺑﺘﺪﺍ Position (ﻣﻮﻗﻌﻴﺖ) ﺭﺍ ﺍﻧﺘﺨﺎﺏ ﮐﺮﺩﻩ ﻭ ﺳﭙﺲ ﺑﺮ ﺭﻭﯼ Set Desktop Background  (ﺗﻨﻈﻴﻢ ﭘﺲ ﺯﻣﻴﻨﻪ ﺩﺳﮑﺘﺎﭖ) ﮐﻠﻴﮏ ﮐﻨﻴﺪ ﻭ  ﮐﺎﺭ ﺗﻤﺎﻡ ﺍﺳﺖ.
 
ﺍﮔﺮ ﻓﺎﻳﻞ ﻋﮑﺲ ﺩﺭ ﺣﺎﻓﻈﻪ ﺩﺳﺘﮕﺎﻩ‌ﺗﺎﻥ ﻗﺮﺍﺭ ﺩﺍﺭﺩ ﻃﺒﻖ ﺭﻭﺵ ﺯﻳﺮ ﻋﻤﻞ ﮐﻨﻴﺪ:
 
ﺍﺑﺘﺪﺍ ﺑﺮ ﺭﻭﯼ ﻋﮑﺲ ﺭﺍﺳﺖ ﮐﻠﻴﮏ ﮐﺮﺩﻩ ﻭ ﺳﭙﺲ ﮔﺰﻳﻨﻪ Set as desktop background (ﺗﻨﻈﻴﻢ ﺑﻪ ﻋﻨﻮﺍﻥ ﭘﺲ ﺯﻣﻴﻨﻪ ﺩﺳﮑﺘﺎﭖ) ﺭﺍ ﺍﻧﺘﺨﺎﺏ ﮐﻨﻴﺪ.
