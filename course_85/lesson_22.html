Save Selection
ﺩﺭ ﺗﻤﺎﻣﯽ ﻣﻮﺍﺭﺩﯼ ﮐﻪ ﺑﺨﺸﯽ ﺍﺯ ﺗﺼﻮﻳﺮ ﺍﻧﺘﺨﺎﺏ ﺷﺪﻩ ﺍﺳﺖ، ﺍﻣﮑﺎﻥ ﺫﺧﻴﺮﻩ ﺳﺎﺯﯼ ﺑﺨﺶ ﺍﻧﺘﺨﺎﺏ ﺷﺪﻩ ﻭﺟﻮﺩ ﺩﺍﺭﺩ. ﺑﺮﺍﯼ ﺍﻳﻦ ﻣﻨﻈﻮﺭ ﺑﺎﻳﺪ ﺑﻪ ﻓﺮﻣﺎﻥ Save Selection، ﻭﺍﻗﻊ ﺩﺭ ﻣﻨﻮﯼ Select ﻣﺮﺍﺟﻌﻪ ﻧﻤﻮﺩ. ﺑﺎ ﮐﻠﻴﮏ ﮐﺮﺩﻥ ﺭﻭﯼ ﺍﻳﻦ ﻓﺮﻣﺎﻥ، ﭘﻨﺠﺮﻩ ﻣﺤﺎﻭﺭﻩ‌ﺍﯼ ﻣﻄﺎﺑﻖ ﺷﮑﻞ ﺯﻳﺮ ﺑﺎﺯ ﻣﯽ‌ﺷﻮﺩ. ﺍﻳﻦ ﻓﺮﻣﺎﻥ ﺩﺭ ﺯﻣﺎﻧﯽ ﮐﻪ ﺍﺑﺰﺍﺭ ﺳﻠﮑﺸﻦ ﻓﻌﺎﻝ ﺍﺳﺖ ﻭ ﻗﺴﻤﺘﯽ ﺍﺯ ﺗﺼﻮﻳﺮ ﺳﻠﮑﺖ ﺷﺪﻩ، ﺑﺎ ﺭﺍﺳﺖ ﮐﻠﻴﮏ ﻧﻴﺰ ﻗﺎﺑﻞ ﺩﺳﺘﺮﺳﯽ ﺍﺳﺖ.

ﺩﺭ ﻣﻨﻮﯼ ﺍﻭﻝ، (Document)، ﮐﺎﺭﺑﺮ ﻣﺸﺨﺺ ﻣﯽ‌ﮐﻨﺪ ﮐﻪ ﻗﺼﺪ ﺩﺍﺭﺩ ﻗﺴﻤﺖ ﺳﻠﮑﺖ ﺷﺪﻩ (Selection) ﺭﺍ ﺩﺭ ﭼﻪ ﺗﺼﻮﻳﺮﯼ ﺫﺧﻴﺮﻩ ﮐﻨﺪ. ﻣﯽ‌ﺗﻮﺍﻥ ﺳﻠﮑﺸﻦ ﺭﺍ ﺩﺭ‌‌ ﻫﻤﺎﻥ ﺗﺼﻮﻳﺮ ﺟﺎﺭﯼ (ﺍﻧﺘﺨﺎﺏ ﻧﺎﻡ ﻓﺎﻳﻞ ﻣﻮﺭﺩ ﻧﻈﺮ)، ﻳﺎ ﺗﺼﻮﻳﺮ ﺩﻳﮕﺮﯼ ﮐﻪ ﺩﺭ ﻣﺤﻴﻂ ﻓﺘﻮﺷﺎﭖ ﺑﺎﺯ ﺍﺳﺖ ﻭ ﺍﺑﻌﺎﺩ ﻳﮑﺴﺎﻧﯽ ﺑﺎ ﺗﺼﻮﻳﺮ ﺟﺎﺭﯼ ﺩﺍﺭﺩ ﻭ ﻳﺎ ﺣﺘﯽ ﻳﮏ ﺗﺼﻮﻳﺮ ﺟﺪﻳﺪ (New، ﺩﺭ ﺍﻳﻦ ﺣﺎﻟﺖ ﻳﮏ ﺗﺼﻮﻳﺮ ﺟﺪﻳﺪ ﺍﻳﺠﺎﺩ ﺷﺪﻩ ﻭ ﻗﺴﻤﺖ ﺍﻧﺘﺨﺎﺏ ﺷﺪﻩ ﺑﻪ ﺭﻧﮓ ﺳﻔﻴﺪ ﺩﺭ ﺗﺼﻮﻳﺮ ﺳﻴﺎﻩ ﻗﺮﺍﺭ ﻣﯽ‌ﮔﻴﺮﺩ) ﺫﺧﻴﺮﻩ ﻧﻤﻮﺩ.

ﻣﻨﻮﯼ Channel، ﺑﻪ ﮐﺎﺭﺑﺮ ﺍﺟﺎﺯﻩ ﻣﯽ‌ﺩﻫﺪ ﮐﻪ ﺳﻠﮑﺸﻦ ﺭﺍ ﺑﻪ ﺻﻮﺭﺕ ﻣﺠﺰﺍ ﺫﺧﻴﺮﻩ ﮐﻨﺪ (New) ﻭ ﻳﺎ ﺍﻳﻨﮑﻪ ﺑﺎ ﻳﮑﯽ ﺍﺯ ﺳﻠﮑﺸﻦ‌ﻫﺎﯼ ﺫﺧﻴﺮﻩ ﺷﺪﻩ ﻗﺒﻠﯽ ﺗﺮﮐﻴﺐ ﻧﻤﺎﻳﺪ (ﺩﺭ ﺍﻳﻦ ﺣﺎﻟﺖ ﺑﺎﻳﺪ ﻧﺎﻡ ﺳﻠﮑﺸﻦ ﻣﻮﺭﺩ ﻧﻈﺮ ﺍﻧﺘﺨﺎﺏ ﮔﺮﺩﺩ). ﺍﺯ ﺍﻳﻦ ﺭﻭ، ﻟﻴﺴﺖ ﺳﻠﮑﺸﻨﻬﺎﻳﯽ ﮐﻪ ﻗﺒﻼ ﺫﺧﻴﺮﻩ ﺷﺪﻩ ﻧﻴﺰ ﺩﺭ ﺍﻳﻦ ﻣﻨﻮ ﺩﻳﺪﻩ ﻣﯽ‌ﺷﻮﺩ.

ﺩﺭ ﻣﻨﻮﯼ Name ﻧﺎﻣﯽ ﺑﺮﺍﯼ ﺍﺧﺘﺼﺎﺹ ﺑﻪ ﺳﻠﮑﺸﻨﯽ ﮐﻪ ﻗﺼﺪ ﺫﺧﻴﺮﻩ ﺳﺎﺯﯼ ﺁﻥ ﺭﺍ ﺩﺍﺭﻳﻢ، ﺍﺯ ﮐﺎﺭﺑﺮ ﺩﺭ ﺧﻮﺍﺳﺖ ﻣﯽ‌ﺷﻮﺩ.

ﻧﮑﺘﻪ: ﻳﮏ ﺳﻠﮑﺸﻦ، ﺩﺭ ﮐﺎﻧﺎﻝ ﺁﻟﻔﺎ (Alpha Channel) ﺫﺧﻴﺮﻩ ﻣﯽ‌ﺷﻮﺩ. ﺑﻨﺎﺑﺮﺍﻳﻦ ﺩﺭ ﺯﻣﺎﻥ ﺫﺧﻴﺮﻩ ﺳﺎﺯﯼ ﻳﮏ ﻓﺎﻳﻞ ﮐﻪ ﺩﺍﺭﺍﯼ ﺳﻠﮑﺸﻦ ﺫﺧﻴﺮﻩ ﺷﺪﻩ ﺍﺳﺖ، ﺑﺎﻳﺪ ﺗﻮﺟﻪ ﺩﺍﺷﺖ ﮐﻪ ﻓﺮﻣﺖ ﺁﻥ ﻓﺎﻳﻞ، ﻗﺎﺑﻠﻴﺖ ﺫﺧﻴﺮﻩ ﮐﺮﺩﻥ ﮐﺎﻧﺎﻝ ﺁﻟﻔﺎ ﺭﺍ ﻧﻴﺰ ﺩﺍﺷﺘﻪ ﺑﺎﺷﺪ. ﻣﺜﻼ ﻓﺮﻣﺖ JPG ﻧﻤﯽ‌ﺗﻮﺍﻧﺪ ﮐﺎﻧﺎﻝ ﺁﻟﻔﺎ ﺭﺍ ﺫﺧﻴﺮﻩ ﮐﻨﺪ ﺑﻪ ﻫﻤﻴﻦ ﺩﻟﻴﻞ ﺩﺭ ﺯﻣﺎﻥ ﺫﺧﻴﺮﻩ ﺳﺎﺯﯼ ﺩﺭ ﻗﺎﻟﺐ JPG، ﺩﺭ ﭘﻨﺠﺮﻩ Save As، ﻋﻼﻣﺖ ﻫﺸﺪﺍﺭﯼ ﻣﻄﺎﺑﻖ ﺷﮑﻞ ﺯﻳﺮ ﺩﺭ ﮐﻨﺎﺭ ﮔﺰﻳﻨﻪ ﮐﺎﻧﺎﻝ ﺁﻟﻔﺎ ﻇﺎﻫﺮ ﻣﯽ‌ﺷﻮﺩ.

ﻓﺮﻣﺖ‌ﻫﺎﻳﯽ ﻣﺎﻧﻨﺪ TIF ﻭ PSD ﻗﺎﺑﻠﻴﺖ ﺫﺧﻴﺮﻩ ﺳﺎﺯﯼ Alpha Channels ﺭﺍ ﺩﺍﺭﺍ ﻫﺴﺘﻨﺪ.

 

Load Selection
ﺑﺮﺍﯼ ﻓﺮﺍﺧﻮﺍﻧﯽ ﺳﻠﮑﺸﻨﻬﺎﯼ ﺫﺧﻴﺮﻩ ﺷﺪﻩ، ﮐﺎﻓﯽ ﺍﺳﺖ ﺑﻪ ﻣﻨﻮﯼ Select ﺭﻓﺘﻪ ﻭ ﻓﺮﻣﺎﻥ Load Selection ﺭﺍ ﺍﻧﺘﺨﺎﺏ ﮐﻨﻴﻢ.

ﺩﺭ ﺍﻳﻦ ﭘﻨﺠﺮﻩ‌‌ ﻫﻤﺎﻥ ﻃﻮﺭ ﮐﻪ ﺩﺭ ﺷﮑﻞ ﺯﻳﺮ ﻧﻴﺰ ﻣﺸﺎﻫﺪﻩ ﻣﯽ‌ﺷﻮﺩ، ﺍﺑﺘﺪﺍ ﺩﺭ ﻣﻨﻮﯼ Document، ﻧﺎﻡ ﻓﺎﻳﻞ ﻣﻮﺭﺩ ﻧﻈﺮ ﺭﺍ ﺍﻧﺘﺨﺎﺏ ﻭ ﺳﭙﺲ ﺩﺭ ﮔﺰﻳﻨﻪ Channel، ﺳﻠﮑﺸﻦ ﻣﻮﺭﺩ ﻧﻈﺮ ﺭﺍ ﺍﻧﺘﺨﺎﺏ ﻣﯽ‌ﮐﻨﻴﻢ.

ﮔﺰﻳﻨﻪ Invert ﺑﺎﻋﺚ ﻣﯽ‌ﺷﻮﺩ ﮐﻪ ﺳﻠﮑﺸﻦ ﺑﺮﻋﮑﺲ ﺷﻮﺩ، ﻳﻌﻨﯽ ﻗﺴﻤﺘﻬﺎﻳﯽ ﮐﻪ ﺳﻠﮑﺖ ﻧﺒﻮﺩﻩ، ﺳﻠﮑﺖ ﻣﯽ‌ﺷﻮﻧﺪ ﻭ ﻗﺴﻤﺘﻬﺎﯼ ﺳﻠﮑﺖ ﺷﺪﻩ، ﺍﺯ ﺣﺎﻟﺖ ﺍﻧﺘﺨﺎﺏ، ﺧﺎﺭﺝ ﻣﯽ‌ﺷﻮﻧﺪ. ﻓﺮﻣﺎﻥ Invert ﺩﺭ ﻣﻨﻮﯼ Select ﻧﻴﺰ ﻗﺮﺍﺭ ﺩﺍﺭﺩ ﮐﻪ ﻭﻇﻴﻔﻪ ﻣﺸﺎﺑﻬﯽ ﺭﺍ ﺩﺭ ﻣﻮﺭﺩ ﺳﻠﮑﺸﻦ ﺟﺎﺭﯼ ﺩﺭ ﺗﺼﻮﻳﺮ ﺑﻌﻬﺪﻩ ﺩﺍﺭﺩ.

ﺩﺭ ﺣﺎﻟﺘﯽ ﮐﻪ ﺗﻤﺎﻳﻞ ﺩﺍﺭﻳﻢ، ﻳﮏ ﺳﻠﮑﺸﻦ ﺭﺍ ﺑﺎ ﺳﻠﮑﺸﻦ ﺩﻳﮕﺮﯼ ﺗﺮﮐﻴﺐ ﮐﻨﻴﻢ، ﺑﺎﻳﺪ ﺍﺑﺘﺪﺍ ﭘﻨﺠﺮﻩ Save Selection ﻭ ﺩﺭ ﻣﻨﻮﯼ Channel، ﺳﻠﮑﺸﻦ ﻣﻮﺭﺩ ﻧﻈﺮ ﺭﺍ ﺍﻧﺘﺨﺎﺏ ﮐﻨﻴﻢ. ﺳﭙﺲ ﻳﮑﯽ ﺍﺯ ﮔﺰﻳﻨﻪ‌ﻫﺎﯼ ﭘﺎﻳﻴﻨﯽ ﭘﻨﺠﺮﻩ ﺭﺍ ﺩﺭ ﺭﺍﺳﺘﺎﯼ ﻣﻨﻈﻮﺭﻣﺎﻥ ﺍﻧﺘﺨﺎﺏ ﻣﯽ‌ﮐﻨﻴﻢ.

- ﺩﺭ ﺯﻣﺎﻧﯽ ﮐﻪ Channel ﺩﺭ ﺣﺎﻟﺖ New ﻗﺮﺍﺭ ﮔﺮﻓﺘﻪ ﺑﺎﺷﺪ، ﺩﺭ ﭘﺎﻳﻴﻦ ﭘﻨﺠﺮﻩ ﻧﻴﺰ ﮔﺰﻳﻨﻪ New Channel ﺍﻧﺘﺨﺎﺏ ﺷﺪﻩ ﮐﻪ ﺑﺎﻋﺚ ﻣﯽ‌ﺷﻮﺩ، ﺳﻠﮑﺸﻦ ﺑﺎ ﻧﺎﻡ ﺟﺪﻳﺪﯼ ﺫﺧﻴﺮﻩ ﺷﻮﺩ.

- ﮔﺰﻳﻨﻪ ﺑﻌﺪﯼ ﻳﻌﻨﯽ Add to Channel، ﺳﻠﮑﺸﻦ ﺟﺪﻳﺪ ﺭﺍ ﺑﻪ ﺳﻠﮑﺸﻦ ﺫﺧﻴﺮﻩ ﺷﺪﻩ ﺍﺿﺎﻓﻪ ﻣﯽ‌ﮐﻨﺪ.

- ﮔﺰﻳﻨﻪ Subtract from Channel، ﺳﻠﮑﺸﻦ ﺟﺪﻳﺪ ﺭﺍ ﺍﺯ ﺳﻠﮑﺸﻦ ﺫﺧﻴﺮﻩ ﺷﺪﻩ ﮐﻢ ﻣﯽ‌ﮐﻨﺪ.

- ﮔﺰﻳﻨﻪ Intersect with Channel، ﻗﺴﻤﺖ ﻣﺸﺘﺮﮎ ﺑﻴﻦ ﺳﻠﮑﺸﻦ ﺟﺪﻳﺪ ﻭ ﺳﻠﮑﺸﻦ ﺫﺧﻴﺮﻩ ﺷﺪﻩ ﺭﺍ ﺑﻪ ﻋﻨﻮﺍﻥ ﺍﻧﺘﺨﺎﺏ ﻧﻬﺎﻳﯽ ﺩﺭ ﻧﻈﺮ ﻣﯽ‌ﮔﻴﺮﺩ.

ﺩﺭ ﺯﻣﺎﻥ ﻟﻮﺩ ﮐﺮﺩﻥ (ﻓﺮﺍﺧﻮﺍﻧﯽ) ﻳﮏ ﺳﻠﮑﺸﻦ ﻧﻴﺰ ﺩﺭ ﭘﻨﺠﺮﻩ ﻣﺮﺑﻮﻃﻪ، ﺑﺎ ﻫﻤﻴﻦ ﮔﺰﻳﻨﻪ‌ﻫﺎ ﻣﻮﺍﺟﻪ ﻫﺴﺘﻴﻢ ﻭ ﻣﯽ‌ﺗﻮﺍﻧﻴﻢ ﺳﻠﮑﺸﻦ ﻟﻮﺩ ﺷﺪﻩ ﺭﺍ ﺑﺎ ﺳﻠﮑﺸﻦ ﺣﺎﺿﺮ ﺗﺮﮐﻴﺐ ﮐﻨﻴﻢ.

 

Elliptical Marquee Tool
ﻳﮑﯽ ﺍﺯ ﺍﺑﺰﺍﺭﻫﺎﯼ ﺩﻳﮕﺮ ﺍﻧﺘﺨﺎﺏ، Elliptical Marquee Tool ﺍﺳﺖ ﮐﻪ ﺩﺭ‌‌ ﻫﻤﺎﻥ ﺟﻌﺒﻪ Rectangular Marquee Tool ﻗﺮﺍﺭ ﺩﺍﺭﺩ ﻭ ﺳﻠﮑﺸﻨﻬﺎﯼ ﺑﻴﻀﯽ ﻳﺎ ﺩﺍﻳﺮﻩ‌ﺍﯼ ﺗﺮﺳﻴﻢ ﻣﯽ‌ﮐﻨﺪ.

ﮐﺎﺭﮐﺮﺩ ﺍﻳﻦ ﺍﺑﺰﺍﺭ ﻣﺎﻧﻨﺪ Rectangular Marquee Tool ﺍﺳﺖ.‌‌ ﻫﻤﺎﻥ ﻃﻮﺭ ﮐﻪ ﻣﺸﺎﻫﺪﻩ ﻣﯽ‌ﺷﻮﺩ، ﺑﺎ ﺗﻮﺟﻪ ﺑﻪ ﻣﻨﺤﻨﯽ ﺑﻮﺩﻥ ﺳﻠﮑﺸﻦ ﺩﺭ ﺍﻳﻦ ﺣﺎﻟﺖ، ﮔﺰﻳﻨﻪ Anti Alias ﻧﻴﺰ ﻓﻌﺎﻝ ﺍﺳﺖ.

 

Single Row ﻳﺎ Single Column Marquee Tool
ﺍﻳﻦ ﺩﻭ ﺍﺑﺰﺍﺭ ﺑﺮﺍﯼ ﺭﺳﻢ ﻳﮏ ﺳﻄﺮ (Single Row) ﻳﺎ ﺳﺘﻮﻥ (Single Column) ﺑﺎ ﻋﺮﺽ ﻳﮏ ﭘﻴﮑﺴﻞ ﺑﻪ ﮐﺎﺭ ﻣﯽ‌ﺭﻭﻧﺪ. ﻳﮑﯽ ﺍﺯ ﮐﺎﺭﺑﺮﺩﻫﺎﯼ ﺍﻳﻦ ﺍﺑﺰﺍﺭ‌ﻫﺎ ﮐﻪ ﺩﺭ ﺟﻌﺒﻪ ﺍﺑﺰﺍﺭ Marquee Tool‌ﻫﺎ ﻗﺮﺍﺭ ﺩﺍﺭﺩ، ﺗﺮﺳﻴﻢ ﻳﮏ ﺳﻄﺮ ﻳﺎ ﺳﺘﻮﻥ ﺩﺭ ﺭﺍﺳﺘﺎﯼ ﺑﺨﺸﯽ ﺍﺯ ﺗﺼﻮﻳﺮ ﺍﺳﺖ ﮐﻪ ﺩﺍﺭﺍﯼ ﺧﻂ ﺭﺍﺳﺖ ﺑﻮﺩﻩ ﻭ ﻣﯽ‌ﺧﻮﺍﻫﻴﻢ ﺁﻥ ﺑﺨﺶ ﺭﺍ ﺳﻠﮑﺖ ﮐﻨﻴﻢ. ﺩﺭ ﺍﻳﻦ ﺣﺎﻟﺖ ﻳﮏ ﺳﻄﺮ ﻳﺎ ﺳﺘﻮﻥ ﺭﺍ ﺍﻳﺠﺎﺩ ﮐﺮﺩﻩ ﻭ ﺳﭙﺲ ﻳﮑﯽ ﺍﺯ ﺍﺿﻼﻉ ﺁﻥ ﺭﺍ ﺗﻮﺳﻂ Transform Selection ﺩﺭ ﺟﻬﺖ ﻣﻨﺎﺳﺐ، ﺩﺭﮒ ﻣﯽ‌ﮐﻨﻴﻢ.

ﺑﺎ ﺗﻮﺟﻪ ﺑﻪ ﺑﺎﺭﻳﮏ ﺑﻮﺩﻥ ﻋﺮﺽ ﺩﺭ ﺍﻳﻦ ﺣﺎﻟﺖ ﺳﻠﮑﺸﻦ، ﮔﺎﻫﯽ ﺑﺮﺍﯼ ﺍﻳﻨﮑﻪ ﻣﺮﺯ ﺳﻠﮑﺸﻦ ﻗﺎﺑﻞ ﺭﻭﻳﺖ ﺑﺎﺷﺪ، ﺑﺎﻳﺪ ﺗﺼﻮﻳﺮ ﺭﺍ ﺑﺰﺭﮔﻨﻤﺎﻳﯽ ﮐﻨﻴﻢ.
